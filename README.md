# StatsRank
Collect the minecraft server stats and rank them.
```
stats_rank 0.1.0
Get stats from world and ranking them

USAGE:
    stats_rank.exe [OPTIONS] --key <KEY>

OPTIONS:
    -e, --exact            The rank key is exactly
    -h, --help             Print help information
    -i, --inverse          Inverse the rank
    -k, --key <KEY>        The stats key to rank
    -l, --limit <LIMIT>    The rank limit for display [default: 9961]
    -p, --path <PATH>      The server path [default: .]
    -s, --show-uuid        Show uuid even the name was found
    -V, --version          Print version information
```
